In [mathematics](/wiki/Mathematics "Mathematics") a transformation of a [sequence's](/wiki/Sequence "Sequence") [generating function](/wiki/Generating_function "Generating function") provides a method of converting the generating function for one sequence into a generating function enumerating another. These transformations typically involve integral formulas applied to a sequence generating function (see [integral transformations](#Integral_transformations)) or weighted sums over the [higher-order derivatives](/wiki/Derivative#Higher-order_derivatives "Derivative") of these functions (see [derivative transformations](#Derivative_transformations)).

Given a sequence, { f n } n \= 0 ∞ {\\displaystyle \\{f\_{n}\\}\_{n=0}^{\\infty }} ![{\displaystyle \{f_{n}\}_{n=0}^{\infty }}](https://wikimedia.org/api/rest_v1/media/math/render/svg/cff7047d8cbca52dd1c08a01edf96377eccc6b0f) , the [ordinary generating function](/wiki/Generating_function "Generating function") (OGF) of the sequence, denoted F ( z ) {\\displaystyle F(z)} ![{\displaystyle F(z)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/5cf298d777120df944559c5e985b88a824debb80) , and the [exponential generating function](/wiki/Generating_function "Generating function") (EGF) of the sequence, denoted F ^ ( z ) {\\displaystyle {\\widehat {F}}(z)} ![{\displaystyle {\widehat {F}}(z)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/83541a41a6b4f28d98215fb8dd68eb38b6a4c518) , are defined by the [formal power series](/wiki/Formal_power_series "Formal power series"):

F ( z ) \= ∑ n \= 0 ∞ f n z n \= f 0 + f 1 z + f 2 z 2 + ⋯ {\\displaystyle F(z)=\\sum \_{n=0}^{\\infty }f\_{n}z^{n}=f\_{0}+f\_{1}z+f\_{2}z^{2}+\\cdots } ![{\displaystyle F(z)=\sum _{n=0}^{\infty }f_{n}z^{n}=f_{0}+f_{1}z+f_{2}z^{2}+\cdots }](https://wikimedia.org/api/rest_v1/media/math/render/svg/a7e671d14644a45243314a18d7680dfa5457ed9e) 

F ^ ( z ) \= ∑ n \= 0 ∞ f n n ! z n \= f 0 0 ! + f 1 1 ! z + f 2 2 ! z 2 + ⋯ . {\\displaystyle {\\widehat {F}}(z)=\\sum \_{n=0}^{\\infty }{\\frac {f\_{n}}{n!}}z^{n}={\\frac {f\_{0}}{0!}}+{\\frac {f\_{1}}{1!}}z+{\\frac {f\_{2}}{2!}}z^{2}+\\cdots .} ![{\displaystyle {\widehat {F}}(z)=\sum _{n=0}^{\infty }{\frac {f_{n}}{n!}}z^{n}={\frac {f_{0}}{0!}}+{\frac {f_{1}}{1!}}z+{\frac {f_{2}}{2!}}z^{2}+\cdots .}](https://wikimedia.org/api/rest_v1/media/math/render/svg/e85124de7172eca230503684c68e02f089123138) 

In this article, we use the convention that the ordinary (exponential) generating function for a sequence { f n } {\\displaystyle \\{f\_{n}\\}} ![{\displaystyle \{f_{n}\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/009619e33115347a277b099ff493347bdd5776aa) is denoted by the uppercase function F ( z ) {\\displaystyle F(z)} ![{\displaystyle F(z)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/5cf298d777120df944559c5e985b88a824debb80) / F ^ ( z ) {\\displaystyle {\\widehat {F}}(z)} ![{\displaystyle {\widehat {F}}(z)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/83541a41a6b4f28d98215fb8dd68eb38b6a4c518) for some fixed or formal z {\\displaystyle z} ![{\displaystyle z}](https://wikimedia.org/api/rest_v1/media/math/render/svg/bf368e72c009decd9b6686ee84a375632e11de98) when the context of this notation is clear. Additionally, we use the bracket notation for coefficient extraction from the _Concrete Mathematics_ reference which is given by \[ z n \] F ( z ) := f n {\\displaystyle \[z^{n}\]F(z):=f\_{n}} ![{\displaystyle [z^{n}]F(z):=f_{n}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/b1dfe645dd8a24dd25cc2796518bd041ae22e984) . The [main article](/wiki/Generating_function "Generating function") gives examples of generating functions for many sequences. Other examples of generating function variants include [Dirichlet generating functions](/wiki/Dirichlet_generating_function "Dirichlet generating function") (DGFs), [Lambert series](/wiki/Lambert_series "Lambert series"), and [Newton series](/wiki/Newton_series "Newton series"). In this article we focus on transformations of generating functions in mathematics and keep a running list of useful transformations and transformation formulas.

## Inversion relations and generating function identities

\[[edit](/w/index.php?title=Generating_function_transformation&action=edit&section=22 "Edit section: Inversion relations and generating function identities")\]

### Inversion relations

\[[edit](/w/index.php?title=Generating_function_transformation&action=edit&section=23 "Edit section: Inversion relations")\]

An _inversion relation_ is a pair of equations of the form

g n \= ∑ k \= 0 n A n , k ⋅ f k ⟷ f n \= ∑ k \= 0 n B n , k ⋅ g k , {\\displaystyle g\_{n}=\\sum \_{k=0}^{n}A\_{n,k}\\cdot f\_{k}\\quad \\longleftrightarrow \\quad f\_{n}=\\sum \_{k=0}^{n}B\_{n,k}\\cdot g\_{k},} ![{\displaystyle g_{n}=\sum _{k=0}^{n}A_{n,k}\cdot f_{k}\quad \longleftrightarrow \quad f_{n}=\sum _{k=0}^{n}B_{n,k}\cdot g_{k},}](https://wikimedia.org/api/rest_v1/media/math/render/svg/02c818c075990733e062e8caff96cc05ab4fd7ec) 

which is equivalent to the _orthogonality relation_

∑ k \= j n A n , k ⋅ B k , j \= δ n , j . {\\displaystyle \\sum \_{k=j}^{n}A\_{n,k}\\cdot B\_{k,j}=\\delta \_{n,j}.} ![{\displaystyle \sum _{k=j}^{n}A_{n,k}\cdot B_{k,j}=\delta _{n,j}.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/170e04fd06da319a564bba57612cae82678f98cd) 

Given two sequences, { f n } {\\displaystyle \\{f\_{n}\\}} ![{\displaystyle \{f_{n}\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/009619e33115347a277b099ff493347bdd5776aa) and { g n } {\\displaystyle \\{g\_{n}\\}} ![{\displaystyle \{g_{n}\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/74958d878c4aec646e8fe4ceb75ce866d6e85f54) , related by an inverse relation of the previous form, we sometimes seek to relate the OGFs and EGFs of the pair of sequences by functional equations implied by the inversion relation. This goal in some respects mirrors the more number theoretic ([Lambert series](/wiki/Lambert_series "Lambert series")) generating function relation guaranteed by the [Möbius inversion formula](/wiki/M%C3%B6bius_inversion_formula "Möbius inversion formula"), which provides that whenever

a n \= ∑ d | n b d ⟷ b n \= ∑ d | n μ ( n d ) a d , {\\displaystyle a\_{n}=\\sum \_{d|n}b\_{d}\\quad \\longleftrightarrow \\quad b\_{n}=\\sum \_{d|n}\\mu \\left({\\frac {n}{d}}\\right)a\_{d},} ![{\displaystyle a_{n}=\sum _{d|n}b_{d}\quad \longleftrightarrow \quad b_{n}=\sum _{d|n}\mu \left({\frac {n}{d}}\right)a_{d},}](https://wikimedia.org/api/rest_v1/media/math/render/svg/b159a4337f27b1dd82de8796dd28073c013eef09) 

the generating functions for the sequences, { a n } {\\displaystyle \\{a\_{n}\\}} ![{\displaystyle \{a_{n}\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/339d461b497fb8f8670bb29308fe09f0e7bfd34a) and { b n } {\\displaystyle \\{b\_{n}\\}} ![{\displaystyle \{b_{n}\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/cad2485b9672375982ec521a53ee5a4104001a15) , are related by the _Möbius transform_ given by

∑ n ≥ 1 a n z n \= ∑ n ≥ 1 b n z n 1 − z n . {\\displaystyle \\sum \_{n\\geq 1}a\_{n}z^{n}=\\sum \_{n\\geq 1}{\\frac {b\_{n}z^{n}}{1-z^{n}}}.} ![{\displaystyle \sum _{n\geq 1}a_{n}z^{n}=\sum _{n\geq 1}{\frac {b_{n}z^{n}}{1-z^{n}}}.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/c00810649a3cba8752a920514018347141e24c91) 

Similarly, the _Euler transform_ of generating functions for two sequences, { a n } {\\displaystyle \\{a\_{n}\\}} ![{\displaystyle \{a_{n}\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/339d461b497fb8f8670bb29308fe09f0e7bfd34a) and { b n } {\\displaystyle \\{b\_{n}\\}} ![{\displaystyle \{b_{n}\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/cad2485b9672375982ec521a53ee5a4104001a15) , satisfying the relation[\[20\]](#cite_note-20)

1 + ∑ n ≥ 1 b n z n \= ∏ i ≥ 1 1 ( 1 − z i ) a i , {\\displaystyle 1+\\sum \_{n\\geq 1}b\_{n}z^{n}=\\prod \_{i\\geq 1}{\\frac {1}{(1-z^{i})^{a\_{i}}}},} ![{\displaystyle 1+\sum _{n\geq 1}b_{n}z^{n}=\prod _{i\geq 1}{\frac {1}{(1-z^{i})^{a_{i}}}},}](https://wikimedia.org/api/rest_v1/media/math/render/svg/e2f74026a260f851ddcc7ed311b3003a69f7deb5) 

is given in the form of

1 + B ( z ) \= exp ⁡ ( ∑ k ≥ 1 A ( z k ) k ) , {\\displaystyle 1+B(z)=\\exp \\left(\\sum \_{k\\geq 1}{\\frac {A(z^{k})}{k}}\\right),} ![{\displaystyle 1+B(z)=\exp \left(\sum _{k\geq 1}{\frac {A(z^{k})}{k}}\right),}](https://wikimedia.org/api/rest_v1/media/math/render/svg/abc2db07119a53f7e77586bbe29b327873a07363) 

where the corresponding inversion formulas between the two sequences is given in the reference.

### The Stirling transform

\[[edit](/w/index.php?title=Generating_function_transformation&action=edit&section=25 "Edit section: The Stirling transform")\]

For any pair of sequences, { f n } {\\displaystyle \\{f\_{n}\\}} ![{\displaystyle \{f_{n}\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/009619e33115347a277b099ff493347bdd5776aa) and { g n } {\\displaystyle \\{g\_{n}\\}} ![{\displaystyle \{g_{n}\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/74958d878c4aec646e8fe4ceb75ce866d6e85f54) , related by the [Stirling number](/wiki/Stirling_number "Stirling number") inversion formula

The _Legendre–Chebyshev classes of inverse relations_ correspond to inversion relations of the form

a n \= ∑ k A n , k ⋅ b k ⟷ b n \= ∑ k B n , k ⋅ ( − 1 ) n − k a k , {\\displaystyle a\_{n}=\\sum \_{k}A\_{n,k}\\cdot b\_{k}\\quad \\longleftrightarrow \\quad b\_{n}=\\sum \_{k}B\_{n,k}\\cdot (-1)^{n-k}a\_{k},} ![{\displaystyle a_{n}=\sum _{k}A_{n,k}\cdot b_{k}\quad \longleftrightarrow \quad b_{n}=\sum _{k}B_{n,k}\cdot (-1)^{n-k}a_{k},}](https://wikimedia.org/api/rest_v1/media/math/render/svg/3ce8584115affb6878c7698ce52e88187dd67c21)

#### Multinomial inverses

a m n \= ∑ j \= 0 m ∑ k \= 0 n ( m j ) ( n k ) ( − 1 ) j + k b j k ⟷ b m n \= ∑ j \= 0 m ∑ k \= 0 n ( m j ) ( n k ) ( − 1 ) j + k a j k , {\\displaystyle a\_{mn}=\\sum \_{j=0}^{m}\\sum \_{k=0}^{n}{\\binom {m}{j}}{\\binom {n}{k}}(-1)^{j+k}b\_{jk}\\quad \\longleftrightarrow \\quad b\_{mn}=\\sum \_{j=0}^{m}\\sum \_{k=0}^{n}{\\binom {m}{j}}{\\binom {n}{k}}(-1)^{j+k}a\_{jk},} ![{\displaystyle a_{mn}=\sum _{j=0}^{m}\sum _{k=0}^{n}{\binom {m}{j}}{\binom {n}{k}}(-1)^{j+k}b_{jk}\quad \longleftrightarrow \quad b_{mn}=\sum _{j=0}^{m}\sum _{k=0}^{n}{\binom {m}{j}}{\binom {n}{k}}(-1)^{j+k}a_{jk},}](https://wikimedia.org/api/rest_v1/media/math/render/svg/9cf93f620325502ea0142c6ff8919d2609617e4e)
